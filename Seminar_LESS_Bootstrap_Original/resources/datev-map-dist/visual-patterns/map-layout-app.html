Basisaufbau fixed-width Layout

Das Layout besteht aus Header und Main-Bereich. Es skaliert bis zu einer maximalen Breite von 960px - danach ist der Content horizontal zentriert.

Beispiel-Anwendung

Überschrift

Content

Hierfür muss dem äußersten Container .dna-page eine feste Breite zugewiesen werden.

HTML
<body class="genepool">
    <div class="dna-page">
        <header class="dna-header">
            ...
        </header>
        <main>
            ...
        </main>
    </div>
</body>
CSS
.dna-page {
    width: 100%;
    max-width: 960px;
    margin: 0 auto;
    overflow: initial;
}

Fluid Layout mit Schublade

Das Layout besteht aus Seite und Schublade. Die Seite enthält Header und Main-Bereich und skaliert beliebig. Die Schublade öffnet sich zu einer festen Breite. Sie schiebt sich bei großen Screengrößen neben die Seite und bei kleinen Screengrößen legt sie sich darüber.

Beispiel-Anwendung

Überschrift

Content

Die Schublade ist eine Angular-Komponente: <datev-sidebar>. Sie wird parallel zur .dna-page in den DOM eingefügt. Das gemeinsame Parent ist der .dna-wrapper, der auch für den dunklen Rand sorgt.

HTML
<body class="genepool">
    <div class="dna-wrapper">
        <datev-sidebar>
            ...
        </datev-sidebar>
        <div class="dna-page">
            ...
        </div>
    </div>
</body>
