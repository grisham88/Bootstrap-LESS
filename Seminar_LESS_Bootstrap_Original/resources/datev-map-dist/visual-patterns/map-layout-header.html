Je nach verwendeter Klasse gibt es verschiedene Standard-Header-Typen. Alle reagieren responsive: Bei kleinen Viewports scrollt der Header mit und wird insgesamt kleiner.

Wichtig: Die folgenden Implementierungen haben Beispiel-Charakter und sind keine festen Vorgaben. Der Header sollte an die Anforderungen der Anwendung angepasst werden.

Code-Basis

Das grundsätzliche Verhalten wird über die Klasse .dna-header zugewiesen. Weitere Klassen für den Inhalt des Headers steuern die Gestaltung innerhalb.

.dna-header-icon
Größe und Postionierung des Programm-Icons
.dna-header-title
Hauptüberschrift, typischerweise eine h2
.dna-header-sub
Weitere Überschrift, kleiner; typischerweise eine h3
.dna-header-function
Container für Funktionen im Header, z.B. Login-Button, Suche,...; Per default rechtsbündig;
.dna-header-breadcrumb
Container für die Breadcrumb
.dna-header-main
Optionaler Container für Icon, Titel und Breadcrumb
.dna-header-item
Generische Klasse für Elemente innerhalb des Headers - verwenden, wenn kein anderes passt.

DNA-Header

Einfachster Einsatz des Headers

Beispiel-Anwendung

Content
HTML
<header class="dna-header">
    <img class="dna-header-icon" src="..." title="..." alt="..."/>
    <h2 class="h2 dna-header-title">Beispiel-Anwendung</h2>
</header>

Anmerkung: Die Einbettung des Icons ist hier nicht optimal und sollte über background-images gelöst werden.

Fixed Header

Fixed Header nur mit Icon und Titel

Beispiel-Anwendung

Content

Soll der Header fest am oberen Ende der Seite hängen, muss der Parent des Headers die Klasse .dna-page-header-fixed erhalten. Zusätzlich muss der Content in einen scrollbaren Container gesetzt werden: .dna-container-scroll. Das Scrollverhalten ändert sich automatisch mit dem Umbruch zum Phone-Breakpoint. Dann scrollt der Header automtisch mit dem Content.

HTML
<div class="dna-page dna-page-header-fixed">
    <header class="dna-header">
        <img src="..." class="dna-header-icon" title="..." alt="..."/>
        <h2 class="h2 dna-header-title">...</h2>
    </header>
    <main>
        <div class="dna-container-scroll"></div>
    </main>
</div>

Komplexe Header-Beispiele

Header können komplexer werden - hier sind einige mögliche Umsetzungen dargestellt. Zum einen kann das Layout durch floats erzeugt werden, zum anderen kann die CSS3-Technik Flexbox verwendet werden - diese wird jedoch vom internet Explorer (fast) gar nciht unterstützt. Daher sollte immer ein entsprechendes Code-Backup gesetzt werden, wenn Flexbox verwendet wird.
